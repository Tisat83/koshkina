Восстановление PIN

Самостоятельно восстановить PIN нельзя — так безопаснее. Восстановление делает администратор, выдавая одноразовую ссылку для вашей квартиры.

Что делать:

  1. Обратитесь к администратору дома.
  2. Администратор создаст одноразовую ссылку для вашей квартиры.
  3. По ссылке вы зададите новый PIN и сможете войти.

Для администратора (если вы сами забыли PIN):
временно включите аварийный вход по телефону, чтобы зайти и создать себе ссылку:
set ALLOW_PHONE_FALLBACK=1 (в этом же окне CMD) и снова запустите python app.py. После восстановления выключите режим: set ALLOW_PHONE_FALLBACK=0.
